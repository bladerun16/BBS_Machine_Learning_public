**Lab 03 – Iris Classification (No-Code)**

This folder contains the no-code reference version of Lab 03, intended to support conceptual understanding during the lab session.

**Files:**

    1: Lab_Classification_DT_final_ncode.ipynb
   Conceptual walkthrough of Iris classification using a Decision Tree model.

    2: iris.csv
   Dataset containing 150 Iris flower samples (Setosa, Versicolor, Virginica).

**Purpose**

This material helps students understand:

    1: The Machine Learning workflow

    2: Decision Tree classification

    3: Training vs. testing accuracy

    4: The basic concept of generalization

**During the Class – Note for Students**

This is a no-code reference version provided for guidance only. You are strongly encouraged to implement the full code independently during practice sessions to strengthen your understanding and develop practical programming skills.
